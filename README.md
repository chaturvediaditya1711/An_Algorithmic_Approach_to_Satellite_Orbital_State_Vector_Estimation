# An_Algorithmic_Approach_to_Satellite_Orbital_State_Vector_Estimation
Developed a MATLAB-based orbit prediction framework for Low Earth Orbit (LEO) satellites using Keplerian propagation, J₂ perturbation modelling, and atmospheric drag analysis with the NRLMSISE-00 model. Validated predicted state vectors against STK simulations to enhance orbit prediction accuracy and support mission planning.
